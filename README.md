# Tutorme.ai
This is Tutorme.ai, an AI based student help tool that guides students through complicated tasks at any age. From elementary, to universities, you can count on this tool for a quick piece of advice, a full on quiz, or even a tutorin session, all done by AI!

### Some standout features:
- **Grade Calibration:** Adapts response complexity (Elementary to University).
- **Interactive Quizzes:** Real-time testing with "typewriter" explanations.

### Technologies used:
- **Frontend:** Next.js, Tailwind CSS, Lucide Icons.
- **Backend:** FastAPI (Python), OpenAI API.

## How to Run Frontend
1. Navigate to the frontend folder and open dedicated terminal: `cd frontend`
2. Install dependencies: `npm install`
3. Start the development server: `npm run dev`
4. PLEASE NOTE THAT I WILL TRY MY BEST TO HOST MY BACKEND AND FRONTEND IN OTHER FREE SERVERS SUCH AS VERCEL AND RENDER INSTEAD OF LOCAL HOST.

## How to Run Backend
1. Open terminal for the backend folder.
2. It should open up the (.venv) environment.
3. Enter "uvicorn main:app --reload" in the terminal to start the backend.
